# Sentiment Analysis using NLP and LSTM

## 📌 Project Description
This project implements a Sentiment Analysis system using Natural Language Processing (NLP) and a Long Short-Term Memory (LSTM) deep learning model to classify social media text into positive, negative, and neutral sentiments.

---

## 🛠 Technologies Used
- Python
- TensorFlow & Keras
- NLP
- Pandas & NumPy
- Scikit-learn
- Google Colab / Jupyter Notebook

---

## 📂 Project Structure
Sentiment_Analysis_LSTM/
│
├── sentiment_dataset.csv
├── sentiment_analysis_lstm.ipynb
├── lstm_model.keras
├── tokenizer.pkl
└── README.md

---

## ⚙️ Model Architecture
- Embedding Layer
- LSTM Layer
- Dropout Layer
- Dense Output Layer

---

## 🚀 How to Run
1. Upload dataset to Google Colab
2. Run the notebook `sentiment_analysis_lstm.ipynb`
3. Train the model
4. Evaluate accuracy and metrics
5. Test predictions with custom input

---

## 📊 Results
The LSTM model achieved good accuracy and successfully classified text sentiments. Performance was evaluated using accuracy, confusion matrix, and classification report.
